# 競合ベンチマーク

        ## 比較対象

        - Atlassian Agile Coach: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- GitHub Skills: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Scrum Guide: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Linear Docs: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。

        ## 採用する基準

        - 入力負荷が低い。
        - 次アクションが明確。
        - ローカルまたは閉域で検証できる。
        - docs と release evidence から状態を追える。

        ## 差別化

        Issue駆動、レビュー、リリース、Codex活用を用語と判断パターンで整理する辞書に絞り、汎用ノートや大型サービスよりも閉域アルファの検証可能性を優先します。
